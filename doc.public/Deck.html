Module Deck

module Deck: sig .. end

suit is the type of suit a playing card can belong to.

type suit =
| Heart
| Diamond
| Spade
| Club 
type value = int

value is the type of point value of a playing card.

type card = {
   suit : suit;
   value : value;
}

card is the type of playing card that holds card suit and value.

type t = card list

t is the type of deck.

val generate_cards_of_suit : suit -> int list -> card list -> card list

generate_cards_of_suit is a list of cards with suit st and values corresponding to ints in the loop list.

val new_deck : t

new_deck is a standard playing card deck with 52 cards.

val shuffle : t -> t

shuffle old_deck is the deck containing all cards from old_deck in shuffled order.

val shuffle_n : t -> int -> t

shuffle_n deck n is the deck containing all cards from deck shuffled n times.

val choose : t -> (card * t) option

choose deck is (card * t) option returning None if deck is empty or Some (card * t) with the first card in deck and the new deck without the first card.

val length : t -> int

length deck is the card count of deck.

val to_list : t -> card list

to_list deck is the list of cards in deck.

val val_of : card -> int

val_of c is the playing value of card c according to Blackjack rules.
